Menu Features


This Pig application features a menu with the following features:

New Game: begin a new game with the same first player (shortcut Ctrl+N)

Toggle Rules: toggle whether rules appear for new games (shortcut Ctrl+T)

Exit: stop the game and end the application upon request (shortcut Ctrl+X)
